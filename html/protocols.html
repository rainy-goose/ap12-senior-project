The Structure of the Internet

Section 1: Protocols


Transmission

There are 2 primary protocols for information transmission on the internet: TCP and UDP. They both have the same fundamental purpose, transmitting data, but drastically differ in methods and potential outcomes.

TCP

TCP stands for Transmission Control Protocol.

UDP

UDP stands for User Datagram Protocol.

Communication

There are many common protocols for communicating over the internet, whether it be with webpages, file transfer, or even as simple as communicating with the router. They all have very different purposes, but are all very important to network design and how the internet came to be.

HTTP/S

HTTP/S stands for HyperText Transfer Protocol (Secure). This is a widely used protocol which enables translation of .html, or HyperText Markup Language, into a working and easily usable website, much like this one.

FTP

FTP stands for File Transfer Protocol. This is often used in file servers, to allow remote clients to access, download, and share files.
